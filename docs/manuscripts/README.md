# Historical Manuscripts

These documents are **not created by this project**. They are historical reference materials included for research purposes.

## Sources & Attribution

### Zuse Internet Archive (ZIA)
The following manuscripts are from the **Konrad Zuse Internet Archive** maintained by the Zuse Institute Berlin (ZIB):
- `ZIA-0367_1941_Logical_Formalisms.pdf` - "Vorbereitung des Plankalküls, logische Formalismen, Schachspiel"
- `ZIA-0368_1941_Chess_Programs.pdf` - "Vorarbeiten zum Plankalkül. Schachprogramme"
- `ZIA-0410_1942_Chess_Programming.pdf` - "Ausarbeitungen zur Programmierung des Schachspiels"
- `Zuse_Theorie_Angewandten_Logistik_Book2.pdf` - Excerpts from "Theorie der angewandten Logistik"

Archive URL: https://zuse.zib.de

### Academic Literature
- `Bruines_2010_Formal_Semantics.pdf` - Bram Bruines' bachelor thesis "Plankalkül: Formal Semantics" (2010), Vrije Universiteit Amsterdam
